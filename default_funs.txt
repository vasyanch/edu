Regular expressions(module re)

pattern - 	шаблон или регулярное выражение,другими словами 
		то что мы ищем с помощью нижележащих функций. Моет содержать как 
		обычные символы так и метасимволы: . ^ $ * + ? { } [ ] \ | ( )
	[abscd] - внутри можно указать множество символов подходящих под наш шаблон 
	[a-zA-Z]   - указываем диапозон
	[^a-zA-Z]   - ^ используется для указания символов кот нам не нужны  
	[a.c] - . точку указываем если хотим найти любой символ(кроме \n)
	'ab*a' - * используется для указания количества поторов символа 
		   после которого ставиться(вкл ноль,
				 + - без нуля, 
			         ? - ноль или одно вхождение
				{n}\{n, m} - указывает кол-во вхождений\диапозон вхождений)
	() - группировка, используется для нахождения повторяющейся
	     группы символов (test)* найдет 'testtest',
	     (test|text)* найдет и 'testtest', и 'testtext', и 'texttext'.
	     \5 - указывает  номер группы которую нужно найти повторно	
	\d - [0-9] 
	\D - [^0-9]
	\s - [ \t\n\r\f\v]
	\S - [^ \t\n\r\f\v]
	\w - [a-zA-Z0-9_]
	\W - [^a-zA-Z0-9_]

Functions from module re
 
re.match(pattern, string) - 	фун., которая проверяет подходит ли 
			    	данная строка под данный шаблон.  
re.search(pattern, string) - 	фун., которая находит первую подстроку 
				которая подходит под наш шаблон  
re.findall(pattern, string) - 	находит все подстроки которые подходят под наш шаблон.
				Возвращает список. 
re.sub(pattern, 'abc', string) - заменяет все вхождения подстрок, которые подходят 
				под наш шаблон чем то другим(2ой аргумент)
